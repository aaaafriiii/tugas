*Deskripsi Proyek
Proyek ini adalah aplikasi sederhana untuk menghitung Indeks Massa Tubuh (BMI) menggunakan input berat badan dan tinggi badan pengguna. Aplikasi ini memberikan klasifikasi berdasarkan hasil BMI yang dihitung, yaitu kekurangan berat badan, berat badan ideal, kelebihan berat badan, dan obesitas.

*Fitur
Menghitung BMI berdasarkan berat badan (kg) dan tinggi badan (cm).
Memberikan klasifikasi BMI sesuai dengan nilai yang dihasilkan.
Memudahkan pengguna dalam memahami status berat badan mereka.

*Prasyarat
Java Development Kit (JDK) versi 8 atau yang lebih baru.
IDE atau editor teks untuk menulis dan menjalankan kode Java (misalnya IntelliJ IDEA, Eclipse, atau Visual Studio Code).

*Cara Menjalankan Proyek
Tinggal RUN

*Contoh output
Masukkan berat badan (kg): 70
Masukkan tinggi badan (cm): 175
BMI Anda adalah: 22.86
Berat badan Anda ideal.
